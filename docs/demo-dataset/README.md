# Demo 数据集说明

本目录存放用于本地演示、联调和评测的样例数据，不属于产品功能文档的一部分，但会被 README、演示脚本和测试流程引用，因此需要单独说明。

## 目录结构

| 路径 | 用途 |
| --- | --- |
| `doc_1.txt` ~ `doc_15.txt` | 小型样例文档，可用于上传和问答演示 |
| `evaluation/` | 评测样本，用于记录问题与期望答案/引用 |
| `adversarial/` | 对抗性样本，用于验证异常输入或坏问题场景 |

## 使用建议

### 演示场景

- 使用 `doc_*.txt` 快速演示上传与问答
- 用较大的外部测试文本验证大文件预览与入库阶段追踪

### 评测场景

- 将 `evaluation/` 视为基线样本集合
- 将 `adversarial/` 视为异常输入或防护样本集合

## 边界说明

- 这些文件主要用于开发、演示和回归验证
- 它们不是正式业务知识库数据
- 不应将它们包装成真实客户数据或生产数据示例

## 对外发布建议

如果你准备将仓库公开发布：

- 保留此目录有助于他人快速验证项目
- 但请避免将任何真实客户文档混入该目录
